Samplepack 3 — Cheetah SpecDrum (TŒRN format)
=============================================

8 voci, WAV mono/16-bit/44100 Hz, una per canale:

  1.wav  KICK        4.wav  HAT (aperto)   7.wav  TOM (mid)
  2.wav  SNARE       5.wav  CLAP           8.wav  COWBELL
  3.wav  HAT (chiuso) 6.wav  TOM (low)

Origine: drum_sklad (Pavel Semiletov) — public domain.
Vedi ../README.md per il set completo e come rigenerarlo.

Nota: i vecchi file _300.._311.wav (kit sintetico, formato NI404) NON vengono
caricati dal firmware TŒRN e possono essere rimossi.
